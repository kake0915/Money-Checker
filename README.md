# Money-Checker

## 概要
日本の紙幣，硬貨が映った画像を入力すると，画像中のお金の合計金額を出力するプログラムである．

出力画像例

## 仕様
・開発環境：Python 3.6.5,OpenCv 3.4.0
・入力画像には黒い背景の上にお金を置いた画像を使用して下さい．
・紙幣，硬貨の裏表は問いません．
・硬貨の模様がはっきり映る（白飛びしない）様に撮影して下さい．
・現時点では1,100,500円玉の判別はできていません．（2018/07/28）
